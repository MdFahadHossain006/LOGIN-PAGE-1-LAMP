# ✨ About The Project

**Premium Lamp Toggle Login Page** is a modern, cinematic login interface inspired by premium UI animations.

Instead of using a normal dark/light mode switch, this project lets users interact with a realistic hanging lamp. Pulling the lamp's string illuminates the entire page with a smooth warm glow, revealing a premium glassmorphism login card with elegant animations.

Designed to deliver a unique first impression, this project combines interactive storytelling with modern frontend technologies.

# 🎮 How It Works

### 🌑 Initial State

- Lamp is OFF
- Dark cinematic background
- Login card hidden
- Ambient shadows only

### 💡 Lamp Toggle

Pull the hanging lamp string.

↓

The lamp swings naturally.

↓

Warm light spreads across the page.

↓

Login form appears with elegant animation.

↓

Pull again to switch OFF.
